#Baneks

Print a anecdote category 'B' from http://baneks.ru to standard output

##Usage:

```bash
banek       #print random banek
banek NUM   #print banek with a number NUM
```

##Installation:

```bash
sudo python setup.py install
```

##Requirements:

* `Python3`
* `requests`
* `lxml`
* `html2text`
