#Baneks

Print a anecdote category 'B' from http://baneks.ru to your console

Usage:

```bash
banek       #print random banek
banek NUM   #print banek with a number NUM
```

Installation:

```bash
python setup.py
```

Requirements:

* `Python3`
* `requests`
* `lxml`

All credits about content belong to baneks.ru
